    Creeate the data_obj[op + "_barcodeGarden"] ("Garden" as opposed to "forest") To save all the trees per barcode or per key. It is an object so that it is easy to just call the right tree given the key. It will be created every time the user wants to group by something different, replacing the previous garden.
